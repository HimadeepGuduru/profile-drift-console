      Connect to Salesforce

      Reads the profile live via the REST API using an access token. Covers system, object, field-level, Apex, Visualforce, custom-permission and tab access.

      Get a token from the Salesforce CLI
      sf org login web -o MyOrg
sf org display -o MyOrg --json
      Copy instanceUrl and accessToken from the JSON output. (Token expires on session timeout — re-run when it does.)

      Live mode needs your site's origin in the org's CORS allowlist (Setup → CORS). App, record-type and layout visibility aren't exposed over REST — use the XML paste path for a complete profile.
